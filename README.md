# 💸 SpendWise

A clean, intuitive budgeting app designed to help users track expenses, manage their money, and build better financial habits — all in one place.

---

## ✨ Overview

SpendWise is a simple yet powerful expense tracking application that allows users to:

* Log daily expenses
* Categorize spending
* View their financial habits clearly
* Stay in control of their budget

Built with a focus on **clean UI**, **usability**, and **real-world functionality**.

---

## 🚀 Features

* 🧾 Add and manage expenses
* 📊 Track spending habits
* 🗂 Categorize transactions
* 💡 Simple and user-friendly interface
* ⚡ Fast and responsive design

---
